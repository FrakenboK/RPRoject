# raw-data/

Эта папка — "кухня" для разработчика, не используется рантаймом.

Здесь могут находиться:
- Исходные выгрузки и дампы (CSV, JSON)
- Промежуточные файлы
- R-скрипты подготовки данных
- Заметки и черновики

**Важно**: Эта папка коммитится в репозиторий, но её содержимое не используется напрямую Shiny приложением или дашбордом.

Для рабочего хранилища данных используйте папку `data/` (не коммитится).
